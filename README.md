# React Native Webview Example

## Library Used
 - https://github.com/awsk1994/React_Native_Webview_Example

## Steps
1. npm install --save react-native-webview
2. Add component:
```js
<WebView
  source={{ uri: 'https://infinite.red' }}
  style={{ marginTop: 20 }}
/>
```
